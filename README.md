# python-script
For downloading BTEUP admit card and results 

# how to use?

## clone the repository
```
git clone https://github.com/Mr-vabs/python-script.git
```

## change the directory
```
cd python-script
```

## install the necessary modules
```
pip install wget html2image
```

## run the code 
- for admit card 
```
python admit.py
```

- for results 
```
python result.py
```
